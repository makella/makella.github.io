Maps

Basemaps

I've been designing basemaps for the web since 2007 using a variety of both proprietary and open source tools. I can work seamlessly between preparing data for cartographic purposes and designing for a multi-scale, online mapping experience. I've designed basemaps for a large software company (Esri) and now for the National Park Service.
